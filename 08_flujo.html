Flujo Eléctrico#

Hasta ahora, hemos calculado el campo eléctrico sumando pacientemente las contribuciones de cada diferencial de carga mediante la Ley de Coulomb. Sin embargo, para distribuciones con alta simetría, existe un “atajo” matemático extraordinariamente poderoso: la Ley de Gauss.

Pero antes de poder usar esta ley, necesitamos entender su concepto fundamental: el Flujo Eléctrico.

La Analogía del Viento#

Imagina que estás en una colina en un día de mucho viento y sostienes una red atrapamariposas (o simplemente un marco cuadrado vacío). Quieres atrapar la mayor cantidad de aire posible que pase a través del marco.

¿De qué depende la “cantidad de viento” que cruza tu marco? Tu intuición te dirá rápidamente que depende de tres cosas:

  1. La velocidad del viento: Mientras más fuerte sople, más aire cruza.

  2. El tamaño del marco: Un marco de 2 metros cuadrados atrapará más aire que uno de 10 centímetros cuadrados.

  3. La orientación: Si pones el marco de cara al viento (perpendicular a la corriente), atraparás el máximo de aire. Si lo giras y lo pones de perfil (paralelo a la corriente), no pasará absolutamente nada de aire a través de él.

En electromagnetismo, el viento invisible es el Campo Eléctrico (\(\vec{E}\)), el tamaño del marco es el Área (\(A\)), y la cantidad total de “campo” que atraviesa esa área es lo que llamamos Flujo Eléctrico (\(\Phi_E\)).

El Vector Área (\(\vec{A}\))#

Para poder calcular esto matemáticamente, necesitamos una forma de representar la “orientación” de nuestra superficie. En física, el área no es solo un escalar (un número), ¡es un vector!

Definimos el vector Área (\(\vec{A}\)) de tal forma que:

  • Su magnitud es igual al tamaño de la superficie (en metros cuadrados).

  • Su dirección es estrictamente perpendicular (normal) a la superficie.

⚠️ Cuidado aquí: Cuando decimos que la superficie está “de cara” al campo eléctrico, en realidad el vector Área \(\vec{A}\) y el vector Campo \(\vec{E}\) son paralelos (el ángulo entre ellos es \(0^\circ\)).

Definición Matemática del Flujo#

Considerando nuestra analogía del viento, la herramienta matemática perfecta para multiplicar dos vectores y medir qué tan “alineados” están es el Producto Punto (o producto escalar).

Para un campo eléctrico uniforme \(\vec{E}\) que atraviesa una superficie plana de área \(\vec{A}\), el flujo eléctrico se define como:

\[\Phi_E = \vec{E} \cdot \vec{A} = E A \cos\theta\]

Donde \(\theta\) es el ángulo entre las líneas de campo y el vector normal a la superficie. Sus unidades en el Sistema Internacional son N·m²/C.

Los tres casos clave:#

  • Flujo Máximo (\(\theta = 0^\circ\)): La superficie está perpendicular al campo. \(\cos(0^\circ) = 1\), por lo tanto \(\Phi_E = E A\).

  • Flujo Nulo (\(\theta = 90^\circ\)): La superficie está de perfil al campo. Las líneas “resbalan” por encima y por debajo sin cruzarla. \(\cos(90^\circ) = 0\), por lo tanto \(\Phi_E = 0\).

  • Flujo Negativo (\(\theta = 180^\circ\)): Las líneas de campo entran por la “cara trasera” de la superficie (en dirección opuesta al vector normal). \(\cos(180^\circ) = -1\), por lo tanto \(\Phi_E = -E A\).

Generalización: La Integral de Flujo#

¿Qué pasa si el campo eléctrico no es uniforme (cambia de un punto a otro) o si nuestra superficie no es plana (como una esfera o un cilindro)?

Aplicamos la misma lógica del cálculo diferencial: dividimos la superficie gigante en infinitos “parches” minúsculos. Cada parche es tan pequeño que se puede considerar plano, con un vector de área diferencial \(d\vec{A}\). El flujo minúsculo que pasa por ese parche es \(d\Phi_E = \vec{E} \cdot d\vec{A}\).

Para hallar el flujo total, sumamos (integramos) todos esos parches sobre toda la superficie \(S\):

\[\Phi_E = \int_{S} \vec{E} \cdot d\vec{A}\]

¡No te asustes con esta integral! Aunque parece temible, pronto descubriremos que, eligiendo inteligentemente la superficie de integración, esta ecuación se resuelve casi sin hacer cálculos matemáticos reales.

            ctx.lineTo(to.x - headlen * Math.cos(angle + Math.PI/7), to.y - headlen * Math.sin(angle + Math.PI/7));
            ctx.lineTo(to.x, to.y);
            ctx.fillStyle = color;
            ctx.fill();
            
            if (label !== "") {
                ctx.fillStyle = color;
                ctx.font = 'italic bold 16px Arial';
                ctx.fillText(label, to.x + 5, to.y - 5);
            }

        function updateSim() {
            let a = parseFloat(slider_a.value);
            let E = parseFloat(slider_E.value);
            let cam_angle = parseFloat(slider_cam.value) * Math.PI / 180;
            
            val_a.innerText = a.toFixed(1);
            val_E.innerText = E.toFixed(1);
            val_cam.innerText = slider_cam.value + "°";
            
            // Actualizar Matemáticas en Leyenda
            let f_val = E * a * a;
            flux_izq.innerText = "-" + f_val.toFixed(1);
            flux_der.innerText = "+" + f_val.toFixed(1);

            ctx.clearRect(0, 0, canvas.width, canvas.height);
            
            // Proyección 3D simple
            let tilt = 0.3; // Inclinación
            function proj3D(local_x, local_y, local_z) {
                // Rotar alrededor de Y
                let rx = local_x * Math.cos(cam_angle) - local_z * Math.sin(cam_angle);
                let rz = local_x * Math.sin(cam_angle) + local_z * Math.cos(cam_angle);
                // Proyectar
                let px = cx + rx * scale3D;
                let py = cy - local_y * scale3D - rz * scale3D * Math.sin(tilt);
                return {x: px, y: py, z: rz}; // Retornar Z para el z-buffer
            }

                proj3D(-a/2, -a/2, -a/2), // 0: izq-aba-atras
                proj3D( a/2, -a/2, -a/2), // 1: der-aba-atras
                proj3D( a/2,  a/2, -a/2), // 2: der-arr-atras
                proj3D(-a/2,  a/2, -a/2), // 3: izq-arr-atras
                proj3D(-a/2, -a/2,  a/2), // 4: izq-aba-frente
                proj3D( a/2, -a/2,  a/2), // 5: der-aba-frente
                proj3D( a/2,  a/2,  a/2), // 6: der-arr-frente
                proj3D(-a/2,  a/2,  a/2)  // 7: izq-arr-frente
            ];

            // Definir las 6 caras con COLORES DISTINTOS y semi-translúcidos
            let faces = [
                { id: 'atras',  pts: [v[0], v[1], v[2], v[3]], center: [0, 0, -a/2], normal: [0, 0, -1], color: 'rgba(150, 150, 255, 0.5)' }, // Azul
                { id: 'frente', pts: [v[4], v[5], v[6], v[7]], center: [0, 0, a/2], normal: [0, 0, 1],   color: 'rgba(200, 100, 255, 0.5)' }, // Violeta
                { id: 'izq',    pts: [v[0], v[3], v[7], v[4]], center: [-a/2, 0, 0], normal: [-1, 0, 0], color: 'rgba(255, 100, 100, 0.7)' }, // Rojo (Entra)
                { id: 'der',    pts: [v[1], v[2], v[6], v[5]], center: [a/2, 0, 0], normal: [1, 0, 0],  color: 'rgba(100, 255, 100, 0.7)' }, // Verde (Sale)
                { id: 'abajo',  pts: [v[0], v[1], v[5], v[4]], center: [0, -a/2, 0], normal: [0, -1, 0], color: 'rgba(255, 180, 100, 0.5)' }, // Naranja
                { id: 'arriba', pts: [v[3], v[2], v[6], v[7]], center: [0, a/2, 0], normal: [0, 1, 0],   color: 'rgba(255, 255, 150, 0.5)' }  // Amarillo
            ];

            // Ordenar caras por profundidad (z_depth) para dibujarlas en orden correcto
            faces.forEach(f => {
                f.z_depth = (f.pts[0].z + f.pts[1].z + f.pts[2].z + f.pts[3].z) / 4;
            });
            faces.sort((a, b) => a.z_depth - b.z_depth);

            // 1. Dibujar Campo Eléctrico de fondo (Flechas grises identificadas con "E")
            ctx.globalAlpha = 0.4; // Ligeramente más visible para leer "E"
            let labelDrawn = false;
            // Dibujamos una cuadrícula de flechas tenues
            for(let y_pos = -2.5; y_pos <= 2.5; y_pos += 1.25) {
                let pStart = proj3D(-7, y_pos, 0); // Flechas a lo largo del plano XZ
                let pEnd = proj3D(7, y_pos, 0);
                
                // Identificar solo una flecha por fila para no saturar
                let labelText = "";
                if(y_pos > 2 && !labelDrawn) {
                    labelText = "E";
                    labelDrawn = true;
                }
                
                drawArrow(pStart, pEnd, 'black', 2, labelText);
            }
            ctx.globalAlpha = 1.0; // Reset

                ctx.moveTo(f.pts[0].x, f.pts[0].y);
                for(let i=1; i<4; i++) ctx.lineTo(f.pts[i].x, f.pts[i].y);
                ctx.closePath();
                ctx.fillStyle = f.color; 
                ctx.fill();
                
                // Bordes
                ctx.strokeStyle = '#333'; ctx.lineWidth = 1;
                // Aristas traseras punteadas

                ctx.stroke(); ctx.setLineDash([]);

                // Dibujar Vector Normal (Rojo)
                let c_proj = proj3D(f.center[0], f.center[1], f.center[2]);
                let norm_len = 1.4; // Largo normal
                let n_proj = proj3D(f.center[0] + f.normal[0]*norm_len, f.center[1] + f.normal[1]*norm_len, f.center[2] + f.normal[2]*norm_len);
                
                // Mostrar normal solo si la cara es visible para no saturar
                if (f.z_depth > -a/4 || f.id === 'izq' || f.id === 'der') {
                    drawArrow(c_proj, n_proj, '#d9534f', 3, "n̂");
                }
            });
        }

        // Eventos
        slider_a.addEventListener('input', updateSim);
        slider_E.addEventListener('input', updateSim);
        slider_cam.addEventListener('input', updateSim);

        updateSim();
