6.3 Reglas de Kirchhoff#

Para circuitos más complejos donde no es posible simplificar únicamente con equivalencias en serie y paralelo, utilizamos las reglas de Kirchhoff.

Regla de los Nodos (Conservación de la Carga)#

En cualquier nodo (punto de unión de tres o más conductores), la suma de las corrientes que entran debe ser igual a la suma de las corrientes que salen.

 

 

Esto implica que no hay acumulación de carga:

\[\sum_{i=1}^{N} I_i = 0\]

Regla de las Mallas (Conservación de la Energía)#

 

 

La suma algebraica de las diferencias de potencial en cualquier trayectoria cerrada (malla) de un circuito debe ser cero:

\[\sum_{i=1}^{N} \mathcal{E}_i - \sum_{j=1}^{N} I_j R_j = 0\]

Ejemplo#

Aplicando las Leyes de Kirchhoff, determine las corrientes del siguiente circuito en función de las resistencias \(R_1\), \(R_2\) y \(R_3\) y de las fuentes de voltaje \(V_1\) y \(V_2\).

 

 

Solución de Circuito con Leyes de Kirchhoff#

1. Identificación de Nodos y Ecuación de Corrientes (Ley de Nodos)

Observando el nodo a, tenemos que las corrientes \(I_1\) e \(I_2\) entran al nodo, mientras que la corriente \(I_3\) sale de él. Aplicando la regla de los nodos:

\[I_1 + I_2 = I_3 \quad \text{(1)}\]

2. Ecuaciones de Mallas (Ley de Voltajes)

Aplicaremos la Segunda Ley de Kirchhoff a las dos mallas principales, recorriéndolas en el sentido de las corrientes dictadas por las fuentes para facilitar el análisis.

  • Malla 1 (Izquierda): Recorriendo en sentido horario desde la fuente \(V_1\). El aumento de potencial de la fuente se iguala a las caídas de potencial en las resistencias:

    \[V_1 - I_1 R_1 - I_3 R_3 = 0 \implies V_1 = I_1 R_1 + I_3 R_3 \quad \text{(2)}\]

  • Malla 2 (Derecha): Recorriendo en sentido antihorario desde la fuente \(V_2\).

    \[V_2 - I_2 R_2 - I_3 R_3 = 0 \implies V_2 = I_2 R_2 + I_3 R_3 \quad \text{(3)}\]

3. Resolución del Sistema de Ecuaciones

Sustituimos la ecuación (1) (\(I_3 = I_1 + I_2\)) en las ecuaciones (2) y (3) para dejar todo en términos de \(I_1\) e \(I_2\):

  • De la ecuación 2:

    \[V_1 = I_1 R_1 + (I_1 + I_2) R_3 \implies V_1 = I_1(R_1 + R_3) + I_2 R_3 \quad \text{(4)}\]

  • De la ecuación 3:

    \[V_2 = I_2 R_2 + (I_1 + I_2) R_3 \implies V_2 = I_1 R_3 + I_2(R_2 + R_3) \quad \text{(5)}\]

Tenemos un sistema lineal de dos ecuaciones (4) y (5) con dos incógnitas. Resolviendo por el método de Cramer o sustitución, primero encontramos el determinante del sistema (\(\Delta\)):

\[\Delta = (R_1 + R_3)(R_2 + R_3) - (R_3)(R_3) = R_1 R_2 + R_1 R_3 + R_2 R_3 + R_3^2 - R_3^2\]
\[\Delta = R_1 R_2 + R_2 R_3 + R_1 R_3\]

Resolviendo para la corriente \(I_1\):

\[I_1 = \frac{V_1(R_2 + R_3) - V_2 R_3}{R_1 R_2 + R_2 R_3 + R_1 R_3}\]

Resolviendo para la corriente \(I_2\):

\[I_2 = \frac{V_2(R_1 + R_3) - V_1 R_3}{R_1 R_2 + R_2 R_3 + R_1 R_3}\]

4. Determinación de \(I_3\)

Finalmente, sustituimos los valores obtenidos de \(I_1\) e \(I_2\) en la Ecuación 1 para encontrar \(I_3\):

\[I_3 = I_1 + I_2 = \frac{V_1(R_2 + R_3) - V_2 R_3 + V_2(R_1 + R_3) - V_1 R_3}{R_1 R_2 + R_2 R_3 + R_1 R_3}\]

Desarrollando y simplificando los términos en el numerador:

\[V_1 R_2 + V_1 R_3 - V_2 R_3 + V_2 R_1 + V_2 R_3 - V_1 R_3 = V_1 R_2 + V_2 R_1\]

Por lo tanto, la expresión final para la corriente \(I_3\) es:

\[I_3 = \frac{V_1 R_2 + V_2 R_1}{R_1 R_2 + R_2 R_3 + R_1 R_3}\]

<script>
(function() {
    const canvas = document.getElementById('canvasKirchhoff_v8');
    if (!canvas) return;
    const ctx = canvas.getContext('2d');

    const sliders = {
        V1: document.getElementById('sliderV1_v8'), V2: document.getElementById('sliderV2_v8'),
        R1: document.getElementById('sliderR1_v8'), R2: document.getElementById('sliderR2_v8'), R3: document.getElementById('sliderR3_v8')
    };
    const labels = {
        V1: document.getElementById('valV1_v8'), V2: document.getElementById('valV2_v8'),
        R1: document.getElementById('valR1_v8'), R2: document.getElementById('valR2_v8'), R3: document.getElementById('valR3_v8'),
        I1: document.getElementById('outI1_v8'), I2: document.getElementById('outI2_v8'), I3: document.getElementById('outI3_v8')
    };

    let currents = [0, 0, 0];
    const particles = [[], [], []];
    const numP = 35; // Densidad de partículas
    const layout = { nTop: {x: 300, y: 50}, nBot: {x: 300, y: 250}, c1x: 100, c2x: 500 };

    function init() {
        for(let g=0; g<3; g++) {
            for(let i=0; i<numP; i++) particles[g].push({pos: Math.random(), r: 2.5});
        }
    }

    function solve() {
        const v1 = parseFloat(sliders.V1.value), v2 = parseFloat(sliders.V2.value);
        const r1 = parseFloat(sliders.R1.value), r2 = parseFloat(sliders.R2.value), r3 = parseFloat(sliders.R3.value);
        
        labels.V1.innerText = v1; labels.V2.innerText = v2;
        labels.R1.innerText = r1; labels.R2.innerText = r2; labels.R3.innerText = r3;

        const det = r1*r2 + r1*r3 + r2*r3;
        const i1 = (v1 * (r2 + r3) - v2 * r3) / det;
        const i2 = (v2 * (r1 + r3) - v1 * r3) / det;
        const i3 = i1 + i2;

        currents = [i1, i2, i3];
        labels.I1.innerText = i1.toFixed(2);
        labels.I2.innerText = i2.toFixed(2);
        labels.I3.innerText = i3.toFixed(2);
    }

    function getPos(branch, t) {
        // Distribución uniforme: cada tramo de 200px ocupa exactamente 1/3 del parámetro 't'
        const third = 1 / 3;
        
        if (branch === 0) { 
            if (t < third) return {x: layout.nBot.x - (t/third)*200, y: layout.nBot.y};
            if (t < 2*third) return {x: layout.c1x, y: layout.nBot.y - ((t-third)/third)*200};
            return {x: layout.c1x + ((t-2*third)/third)*200, y: layout.nTop.y};
        }
        if (branch === 1) { 
            if (t < third) return {x: layout.nBot.x + (t/third)*200, y: layout.nBot.y};
            if (t < 2*third) return {x: layout.c2x, y: layout.nBot.y - ((t-third)/third)*200};
            return {x: layout.c2x - ((t-2*third)/third)*200, y: layout.nTop.y};
        }
        // La rama central es solo un tramo de 200px
        return {x: layout.nTop.x, y: layout.nTop.y + t*200}; 
    }

    function drawSource(x, y, txt, vert) {
        ctx.save();
        ctx.translate(x, y);
        if (!vert) ctx.rotate(Math.PI / 2);
        ctx.fillStyle = '#ffffff'; ctx.fillRect(-15, -4, 30, 7);
        ctx.strokeStyle = '#000'; ctx.lineWidth = 2;
        ctx.beginPath(); ctx.moveTo(-15, -5); ctx.lineTo(15, -5); ctx.stroke();
        ctx.lineWidth = 4;
        ctx.beginPath(); ctx.moveTo(-8, 5); ctx.lineTo(8, 5); ctx.stroke();
        ctx.fillStyle = '#000'; ctx.font = 'bold 14px Arial'; ctx.lineWidth = 1;
        ctx.fillText(txt, 20, 5);
        ctx.restore();
    }

    function drawResistor(x, y, txt, vert) {
        ctx.save();
        ctx.translate(x, y);
        if (vert) ctx.rotate(Math.PI / 2);
        ctx.fillStyle = '#ffffff'; ctx.fillRect(-20, -10, 40, 20);
        ctx.strokeStyle = '#000'; ctx.lineWidth = 2; ctx.beginPath();
        ctx.moveTo(-20, 0);
        ctx.lineTo(-15, -8); ctx.lineTo(-10, 8);
        ctx.lineTo(-5, -8);  ctx.lineTo(0, 8);
        ctx.lineTo(5, -8);   ctx.lineTo(10, 8);
        ctx.lineTo(15, -8);  ctx.lineTo(20, 0);
        ctx.stroke();
        ctx.fillStyle = '#000'; ctx.font = 'bold 14px Arial'; ctx.lineWidth = 1;
        if (vert) { ctx.rotate(-Math.PI / 2); ctx.fillText(txt, 15, 5); }
        else { ctx.fillText(txt, -8, -15); }
        ctx.restore();
    }

    function render() {
        ctx.clearRect(0,0,600,300);
        
        ctx.strokeStyle = '#333'; ctx.lineWidth = 2;
        ctx.strokeRect(layout.c1x, layout.nTop.y, 200, 200);
        ctx.strokeRect(layout.nTop.x, layout.nTop.y, 200, 200);
        ctx.fillStyle = '#333';
        ctx.beginPath(); ctx.arc(layout.nTop.x, layout.nTop.y, 5, 0, 7); ctx.fill();
        ctx.beginPath(); ctx.arc(layout.nBot.x, layout.nBot.y, 5, 0, 7); ctx.fill();

        // LÓGICA DE VELOCIDAD ESTRICTA (Px/seg normalizado)
        const maxAbsoluteCurrent = Math.max(
            Math.abs(currents[0]), 
            Math.abs(currents[1]), 
            Math.abs(currents[2])
        );

        const colors = ['#dc3545', '#198754', '#0d6efd'];
        const pathLengths = [600, 600, 200]; // Longitudes reales en píxeles de cada recorrido
        
        const maxPixelSpeed = 3.0; // Píxeles por frame para la corriente máxima
        const minPixelSpeed = 0.2; // Píxeles por frame mínimos (para que no parezca roto si la I es minúscula)

        particles.forEach((g, i) => {
            ctx.fillStyle = colors[i];
            
            let currentScale = 0;
            if (maxAbsoluteCurrent > 0) {
                currentScale = Math.abs(currents[i]) / maxAbsoluteCurrent;
            }
            
            let pixelSpeed = currentScale * maxPixelSpeed;
            // Si la corriente matemática es distinta de cero (con un pequeño margen de error flotante), asegurar que se mueva algo.
            if (Math.abs(currents[i]) > 0.001) {
                pixelSpeed = Math.max(minPixelSpeed, pixelSpeed);
            } else {
                pixelSpeed = 0;
            }
            
            // Aquí dividimos la velocidad de píxeles por la longitud del cable para obtener el incremento paramétrico correcto.
            let particleFlow = (pixelSpeed / pathLengths[i]) * Math.sign(currents[i]);
            
            g.forEach(p => {
                p.pos += particleFlow;
                if(p.pos > 1) p.pos = 0; if(p.pos < 0) p.pos = 1;
                let pt = getPos(i, p.pos);
                ctx.beginPath(); ctx.arc(pt.x, pt.y, p.r, 0, 7); ctx.fill();
            });
        });

        drawSource(layout.c1x, 150, "V1", true);
        drawSource(layout.c2x, 150, "V2", true);
        drawResistor(200, layout.nTop.y, "R1", false);
        drawResistor(400, layout.nTop.y, "R2", false);
        drawResistor(layout.nTop.x, 150, "R3", true);

        requestAnimationFrame(render);
    }

    Object.values(sliders).forEach(s => s.oninput = solve);
    init(); solve(); render();
})();
</script>
