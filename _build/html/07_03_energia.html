7.3 Energía Potencial Magnética#

Como vimos en la sección anterior, una espira inmersa en un campo magnético uniforme experimenta un momento de torsión (torque) que tiende a alinear su momento dipolar magnético \(\vec{\mu}\) con el campo \(\vec{B}\). La presencia de este torque implica que existe un trabajo mecánico asociado a la rotación de la espira.

Recordemos de la mecánica clásica que el trabajo infinitesimal \(dW\) realizado por un torque \(\tau\) para rotar un objeto un ángulo \(d\theta\) está dado por:

\[dW = -\tau d\theta\]

El signo negativo indica que el torque magnético realiza un trabajo que tiende a disminuir el ángulo \(\theta\) entre \(\vec{\mu}\) y \(\vec{B}\) (es decir, busca la alineación).

Si sustituimos la magnitud del torque magnético, \(\tau = \mu B \sin\theta\), en nuestra expresión de trabajo obtenemos:

\[dW = -(\mu B \sin\theta) d\theta\]

Al igual que en electrostática o en gravitación, el trabajo realizado por una fuerza o torque conservativo es igual al cambio negativo en la energía potencial (\(dU = -dW\)). Por lo tanto, el cambio infinitesimal en la energía potencial magnética \(dU\) es:

\[dU = \mu B \sin\theta d\theta\]

Para encontrar la energía potencial macroscópica \(U(\theta)\), integramos esta expresión. Por convención, definimos el nivel de referencia de la energía potencial (\(U = 0\)) cuando el momento magnético es perpendicular al campo magnético (\(\theta = \pi/2\)). Integramos entonces desde este ángulo de referencia hasta un ángulo arbitrario \(\theta\):

\[U(\theta) - U(\pi/2) = \int_{\pi/2}^{\theta} \mu B \sin\theta' d\theta'\]
\[U(\theta) - 0 = \mu B \left[ -\cos\theta' \right]_{\pi/2}^{\theta}\]
\[U(\theta) = -\mu B \cos\theta\]

Reconociendo que \(\mu B \cos\theta\) es la definición geométrica del producto escalar (o producto punto) entre dos vectores, obtenemos la expresión general para la energía potencial magnética de un dipolo:

\[U = -\vec{\mu} \cdot \vec{B}\]

Análisis de los estados de energía#

La ecuación \(U = -\vec{\mu} \cdot \vec{B}\) nos permite identificar dos posiciones críticas para el momento dipolar en el campo magnético:

  • Energía Mínima (Equilibrio Estable): Ocurre cuando \(\vec{\mu}\) y \(\vec{B}\) son paralelos (\(\theta = 0^\circ\)). En este punto, \(\cos(0) = 1\), por lo que \(U_{\text{mín}} = -\mu B\). La espira está en su configuración de menor energía y cualquier pequeña perturbación generará un torque restaurador que la hará oscilar alrededor de este punto.

  • Energía Máxima (Equilibrio Inestable): Ocurre cuando \(\vec{\mu}\) y \(\vec{B}\) son antiparalelos (\(\theta = 180^\circ\) o \(\pi\)). Aquí, \(\cos(\pi) = -1\), por lo que \(U_{\text{máx}} = +\mu B\). Aunque el torque neto en este punto exacto es cero, la espira está en el pico de su energía potencial; una perturbación infinitesimal provocará un torque que tenderá a girarla fuertemente hacia la posición estable.

Finalmente, el trabajo que debe realizar un agente externo para rotar el dipolo desde un ángulo inicial \(\theta_i\) hasta un ángulo final \(\theta_f\) a velocidad constante es igual a la diferencia de energía potencial:

\[W_{\text{ext}} = \Delta U = U_f - U_i = (-\vec{\mu} \cdot \vec{B}_f) - (-\vec{\mu} \cdot \vec{B}_i)\]

            function drawArrow(ctx, x, y, dx, dy, color, lineWidth=2) {
                const headlen = 10; const angle = Math.atan2(dy, dx);
                ctx.beginPath(); ctx.moveTo(x, y); ctx.lineTo(x + dx, y + dy);
                ctx.strokeStyle = color; ctx.lineWidth = lineWidth; ctx.stroke();
                ctx.beginPath();
                ctx.moveTo(x + dx, y + dy);
                ctx.lineTo(x + dx - headlen * Math.cos(angle - Math.PI / 6), y + dy - headlen * Math.sin(angle - Math.PI / 6));

                const cx = cvsSys.width / 2;
                const cy = cvsSys.height / 2;
                
                ctxS.strokeStyle = '#a0aabc'; ctxS.lineWidth = 1;
                for(let y = 30; y <= cvsSys.height; y += 40) {
                    drawArrow(ctxS, 10, y, cvsSys.width - 30, 0, '#a0aabc', 1.5);
                }
                ctxS.fillStyle = '#a0aabc'; ctxS.font = '14px Arial'; ctxS.fillText('Campo B', cvsSys.width - 65, 20);
                
                const loopRadius = 70;
                const loopX = loopRadius * Math.cos(theta + Math.PI/2);
                const loopY = loopRadius * Math.sin(theta + Math.PI/2);
                
                ctxS.beginPath();
                ctxS.moveTo(cx - loopX, cy + loopY);
                ctxS.lineTo(cx + loopX, cy - loopY);
                ctxS.strokeStyle = '#cd7f32'; ctxS.lineWidth = 6; ctxS.lineCap = 'round';
                ctxS.stroke();
                
                ctxS.beginPath(); ctxS.arc(cx, cy, 5, 0, Math.PI*2);
                ctxS.fillStyle = '#333'; ctxS.fill();
                
                const muLen = 80;
                const mux = muLen * Math.cos(theta);
                const muy = muLen * -Math.sin(theta);
                drawArrow(ctxS, cx, cy, mux, muy, '#0d6efd', 4);
                ctxS.fillStyle = '#0d6efd'; ctxS.font = 'bold 16px sans-serif';
                ctxS.fillText('μ', cx + mux + 10, cy + muy + 10);
